# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-XX

### Added
- Initial release of Fin - Currency & Crypto Converter + Calculator
- Currency converter with real-time exchange rates
- Support for 10+ fiat currencies and 10+ cryptocurrencies
- Cross-conversion between fiat and crypto
- Full-featured calculator with basic arithmetic operations
- Zustand state management for both converter and calculator
- Favorites system with AsyncStorage persistence
- Conversion history (up to 50 conversions)
- Auto-conversion with smart debouncing
- Search functionality in currency selector
- Network status detection and offline mode
- Toast notification system
- Haptic feedback throughout the app
- Responsive design for portrait and landscape orientations
- Error boundaries for graceful error handling
- Loading states and skeleton components
- Accessibility labels and ARIA support
- Performance optimizations (React.memo, useCallback, useMemo)
- Beautiful animations and transitions

### Technical
- TypeScript throughout for type safety
- React Navigation with bottom tabs
- Expo SDK 54
- Modern React patterns and hooks
- API integration with caching strategy
- Comprehensive error handling
